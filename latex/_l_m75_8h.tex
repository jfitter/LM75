\subsection{Detailed Description}
\hyperlink{class_l_m75}{L\-M75} Family Device Driver Library -\/ C\-P\-P Header file. Based on the Melexis \hyperlink{class_l_m75}{L\-M75} Family Data Sheet 3901090614 Rev 004 09jun2008. \begin{DoxyItemize}
\item The current implementation does not manage P\-W\-M (only digital data by I2\-C). \item Sleep mode is not implemented yet.\end{DoxyItemize}
\begin{DoxyNote}{Note}
T\-H\-I\-S I\-S O\-N\-L\-Y A P\-A\-R\-T\-I\-A\-L R\-E\-L\-E\-A\-S\-E. P\-L\-E\-A\-S\-E K\-E\-E\-P T\-H\-I\-S I\-N M\-I\-N\-D I\-F Y\-O\-U D\-E\-C\-I\-D\-E T\-O U\-S\-E T\-H\-I\-S P\-A\-R\-T\-I\-C\-U\-L\-A\-R C\-O\-D\-E F\-O\-R A\-N\-Y\-T\-H\-I\-N\-G.
